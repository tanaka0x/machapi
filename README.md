# XXコンをスクレイピングするやつ

## 対応済み
* オタパ
* koikoi

## dev

**install**
* yarn
* [create-elm-app](https://github.com/halfzebra/create-elm-app)


**start crawling script**
```sh
$ bash crawl.sh
```

**start web server**
```sh
$ node api.js
```


**start dev server**
```sh
$ cd front
$ elm-app start
```

**build assets**
```sh
$ cd front
$ elm-app build
```
